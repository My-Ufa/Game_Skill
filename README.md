# Game_Skill — Claude Code Plugin Marketplace

Маркетплейс плагинов для **Claude Code** по методологии ситуационно-деятельностных игр (СД-игр).

## Как установить

В Claude Code выполни:

```
/plugin install https://github.com/My-Ufa/Game_Skill
```

---

## Плагины

### 🎮 Конструктор СД-игры (`sd-game-builder`)

Опытный игротехник по методологии СД-игр — помогает собрать экономическую игру с нуля за **8 шагов**.

#### Шаги сборки

| # | Шаг | Результат |
|---|-----|-----------|
| 🎤 1 | Транскрибация встречи | Структурированный текст → блоки СД-игры |
| 📊 2 | Анализ рынка | Реальные цены × объёмы = оборот ₽ → у.е. |
| 📝 3 | Описание игры | 8 блоков: контекст / конфликт / механики / гипотезы |
| 🎭 4 | Состав ролей | 10–15 реальных российских компаний отрасли |
| 💰 5 | Экономика | Сценарная таблица MIN/MAX × 3 такта → Excel |
| 🗺️ 6 | Плацдарм | Визуальная схема на A4 → PPTX |
| 📄 7 | Ролевые листы | Индивидуальный лист на роль → PPTX |
| 🎯 8 | Установка на игру | Презентация 14 слайдов → PPTX |

#### Методология

- **Роли** — реальные российские компании (Сбербанк, Яндекс, X5 Group, ЦБ РФ и т.п.)
- **Состав:** 1–2 регулятора, 1–2 банка/инвестора, 1–2 тех.компании, остальное — бизнесы
- **Экономика:** считается в ₽ по реальным ценам, конвертируется в у.е. через делитель
- **Баланс тактов:** такт 2 при бездействии ≈ 0, такт 3 = минус (давление на действие)
- **Открытые вопросы:** после каждого шага — конкретные пробелы и уточнения

#### Как активировать скилл

Скажи Claude Code:
- «Помоги собрать СД-игру»
- «Хочу разработать игру под [отрасль]»
- «Анализ рынка для деловой игры»
- «Состав ролей для игры»
- «Установка на игру»
