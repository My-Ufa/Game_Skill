# Шаг 6 — Плацдарм (→ PPTX, ВСЕ РОЛИ НА ОДНОМ слайде)

## Цель

Все роли на **одном** слайде — компактная сетка карточек.
Генератор автоматически подбирает размер: 4 колонки (≤12 ролей) или 5 колонок (>12).

## Цвет плашки по типу роли

| Тип | Цвет плашки |
|-----|-------------|
| Регулятор | 🔴 Красный |
| Банк | 🟣 Фиолетовый |
| Технологическая компания | 🔵 Синий |
| Бизнес | 🟢 Зелёный |

## Карточка роли содержит

- **Название** (жирно, адаптивный размер)
- Тег (курсив, цвет типа)
- 💰 Стартовый счёт (у.е.)
- − Постоянные расходы/такт (у.е.)

## Что проверить перед генерацией

Для каждой роли должны быть заполнены:
- `name` — название
- `tag` — тип (Регулятор / Банк / Технологическая компания / Бизнес)
- `starting_cash` — стартовый счёт (у.е.)
- `fixed_costs` — постоянные расходы/такт (у.е.)

Если что-то пустое — дозаполни перед нажатием «Сгенерировать».

## Что проверить перед генерацией

- Данные по всем ролям заполнены (счёт, расходы, активы)?
- Цепочка стрелками показана (если она есть)?
- Клиенты размещены снизу?
- Есть ли таблица по тактам для клиентов (если объём меняется)?
