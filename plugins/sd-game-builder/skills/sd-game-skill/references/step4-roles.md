# Шаг 4 — Состав ролей

## Цель

Подобрать 10–15 ролей — реальных российских компаний отрасли.
Каждая роль должна быть узнаваемой и отражать реальную функцию компании.

## Требования к ролям

**Количество:** 10–15 (не меньше 10, не больше 15).

**Каждая роль — реальная российская компания**, например:
- Регуляторы: ЦБ РФ, ФАС, Роскомнадзор, Минпромторг, Росстандарт, Роспотребнадзор
- Банки/Инвесторы: Сбербанк, ВТБ, Альфа-Банк, Тинькофф, РФПИ, ВЭБ.РФ
- Тех.компании: Яндекс, 1С, VK, Сбер AI, МТС AI, Positive Technologies, К2Тех
- Бизнесы: X5 Group, Магнит, Wildberries, Ozon, Газпром нефть, Северсталь,
  Татнефть, Ростелеком, Аэрофлот — в зависимости от отрасли

**Никаких абстрактных имён**: не «Производитель №1», а конкретная компания.

## Строгое распределение по типам

| Тип | Количество | Примеры |
|-----|-----------|---------|
| Регулятор | 1–2 | ЦБ РФ, ФАС, профильное министерство |
| Банк / Инвестор | 1–2 | Сбербанк, ВТБ, РФПИ |
| Технологическая компания | 1–2 | Яндекс, 1С, МТС AI |
| Бизнес | остальное (6–10) | основные игроки отрасли |

## Цепочка продукта

Необязательно покрывать все звенья отдельными ролями.
Одна роль может производить продукт целиком — не создавай лишних ролей
ради «полноты цепочки».

## Для каждой роли укажи

- **Название** — реальная компания
- **Тег** — Регулятор / Банк / Технологическая компания / Бизнес
- **Функция** — 1 строка: что эта компания реально делает в отрасли

## Балансировка

Ни одна роль не должна доминировать на старте: похожие стартовые возможности,
интересные механики у каждого.
