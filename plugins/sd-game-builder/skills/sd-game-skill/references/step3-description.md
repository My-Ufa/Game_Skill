# Шаг 3 — Описание игры

## Цель

Собрать единый документ-описание СД-игры по 8 блокам.
Он станет основой для генерации экономики, плацдарма, ролевых листов и установки.

## Структура описания

### 1. Контекст и ситуация
Отрасль, регион, временной горизонт, ключевая проблема/возможность рынка.

### 2. Основной конфликт
Что сталкивается: интересы каких сторон, какой ресурс ограничен,
чьё положение под угрозой.

### 3. Цепочка создания продукта
Производство → Дистрибуция → Конечный потребитель (или вариант под отрасль).
**Важно:** цепочка может быть укороченной — одна роль способна производить
продукт целиком (вертикально интегрированный игрок). Не вытягивай звенья
искусственно.

### 4. Сценарии и события
Что может произойти за 3 такта:
внешние шоки / действия регулятора / выход технологии / изменение спроса.

### 5. Механики
- **Базовые:** покупка/продажа, найм, строительство, кредитование
- **Технологические:** ИИ, роботизация — что меняют (затраты / персонал /
  скорость / качество / охват рынка)
- **Регуляторные:** лицензии, штрафы, постановления
- **Финансовые:** кредиты, дотации (с условиями), инвестиции

### 6. Временной горизонт
Обычно 3 такта. 1 такт = N лет — так, чтобы был виден эффект от инвестиций
в технологии.

### 7. Гипотезы для проверки игрой
2–4 вопроса формата «Что произойдёт, если…».

### 8. Победа / результат
Как оценивается успех каждой стороны.

## Правила ведения диалога

- Задавай по 1–2 уточняющих вопроса за раз — не вываливай всё сразу.
- Опирайся на материалы из шагов 1 и 2, если они были.
- После каждого ответа пользователя обнови GAME_UPDATE.

## Блок «❓ Открытые вопросы» (обязательно в каждом ответе)

Явно называй, что ещё не описано, противоречит другим блокам или вызывает сомнения.
Вопросы должны быть **конкретными и адресными**, не общими.

Пример:
> ❓ Открытые вопросы:
> 1. Сколько лет в одном такте? Нужно понять, виден ли эффект от R&D за 1 такт.
> 2. Регулятор только штрафует или также выдаёт лицензии? Это влияет на механику.
> 3. Клиенты — отдельная роль или поле спроса (цифра на плацдарме)?
