# Шаг 8 — Установка на игру (→ PPTX, 15 слайдов)

## Цель

Создать презентацию для ввода игроков в игру.
Показывается перед началом: объясняет контекст, правила, роли, экономику.

## Структура (15 слайдов строго)

| # | Слайд | Содержание |
|---|-------|------------|
| 1 | **Титул** | Название игры, отрасль, ведущий, дата |
| 2 | **Контекст рынка** | Ключевые цифры из шага 2 (оборот, CAGR, доли) |
| 3 | **Основной конфликт** | Что сталкивается, за что борьба, чьё положение под угрозой |
| 4 | **Плацдарм** | Схема из шага 6 (вставить как изображение) |
| 5 | **Состав ролей** | Список всех ролей с тегами и функциями |
| 6 | **Цепочка создания продукта** | Кто → кому → что, стрелками |
| 7 | **Механика рынка** | Как разыгрывается спрос (см. ниже) |
| 8 | **Игровые предметы** | Таблица всех активов (см. ниже) |
| 9 | **Ключевые механики** | Базовые / Регуляторные / Финансовые |
| 10 | **Технологии** | Таблица технологий (см. ниже) |
| 11 | **Экономика** | Валюта (у.е. → ₽), мин/макс заработок, общий баланс |
| 12 | **Правила хода** | 1 такт = N лет; всего 3 такта; что происходит внутри такта |
| 13 | **Заявки и документы** | Платёжное поручение / Договор / Заявка на рынок / Постановление регулятора |
| 14 | **Цели сторон** | Для каждой роли — как оценивается победа |
| 15 | **Гипотезы и рефлексия** | Что проверяем игрой; что обсуждаем после |

## Слайд 7 — Механика рынка (разверни полностью)

```
МЕХАНИКА РЫНКА
──────────────────────────────────────────
Правило распределения клиентов:
  [по скорости / по цене / по качеству / по доле / смешанное]

Как подаётся заявка:
  — Форма / порядок действий в такте
  — Кто и когда подаёт

Что происходит при избытке спроса:
  — [очередь следующего такта / потеря клиентов / надбавка к цене]

Что происходит при нехватке предложения:
  — [дефицит фиксируется / регулятор реагирует / цена растёт]
```

Это ключевой слайд — без понятной механики рынка игроки не знают,
зачем покупать предметы и технологии.
