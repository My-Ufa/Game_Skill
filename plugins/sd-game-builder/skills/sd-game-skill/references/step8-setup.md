# Шаг 8 — Установка на игру (→ PPTX, 14 слайдов)

## Цель

Создать презентацию для ввода игроков в игру.
Показывается перед началом: объясняет контекст, правила, роли, экономику.

## Структура (14 слайдов строго)

| # | Слайд | Содержание |
|---|-------|------------|
| 1 | **Титул** | Название игры, отрасль, ведущий, дата |
| 2 | **Контекст рынка** | Ключевые цифры из шага 2 (оборот, CAGR, доли) |
| 3 | **Основной конфликт** | Что сталкивается, за что борьба, чьё положение под угрозой |
| 4 | **Плацдарм** | Схема из шага 6 (вставить как изображение) |
| 5 | **Состав ролей** | Список всех ролей с тегами и функциями |
| 6 | **Цепочка создания продукта** | Кто → кому → что |
| 7 | **Ключевые механики** | Базовые / Технологические / Регуляторные / Финансовые |
| 8 | **Технологии и эффекты** | Таблица: технология → эффект → стоимость → окупаемость |
| 9 | **Экономика** | Валюта (у.е. → ₽), мин/макс заработок, общий баланс |
| 10 | **Правила хода** | 1 такт = N лет; всего 3 такта; что происходит внутри такта |
| 11 | **Заявки и документы** | Платёжное поручение / Договор / Заявка на рынок / Постановление регулятора |
| 12 | **Цели сторон** | Для каждой роли — как оценивается победа |
| 13 | **Гипотезы** | 2–4 «Что произойдёт, если…» — что проверяем игрой |
| 14 | **Финал — рефлексия** | Что обсуждаем после игры: вопросы, сравнение с реальностью |

## Что проверить перед генерацией

Пройдись по чеклисту — если есть пробелы, назови их:

- [ ] Название игры и отрасль установлены (слайд 1)?
- [ ] Есть цифры рынка из шага 2 (слайд 2)?
- [ ] Конфликт сформулирован (слайд 3)?
- [ ] Плацдарм готов (слайд 4)?
- [ ] Все роли подтверждены, 10–15 штук (слайд 5)?
- [ ] Цепочка определена (слайд 6)?
- [ ] Механики прописаны все 4 типа (слайд 7)?
- [ ] Технологии рассчитаны (слайд 8)?
- [ ] Экономика сбалансирована, у.е./₽ зафиксировано (слайд 9)?
- [ ] Правила хода: длина такта в годах, что делают игроки (слайд 10)?
- [ ] Определены типы заявок (слайд 11)?
- [ ] Цели сторон прописаны (слайд 12)?
- [ ] Гипотезы сформулированы (слайд 13)?
- [ ] Вопросы для рефлексии есть (слайд 14)?
